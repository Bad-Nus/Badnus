集合

一、性质与关系

- 性质：①确定性 ②互异性 ③无序性
- 关系：元素 $\in$ 集合，子集 $\subseteq$ 集合，真子集 $\subsetneqq$ 集合

二、常见数集（无限集）

- 正整数：$\mathbb{N}^+$ 或 $\mathbb{N}^*$
- 非负整数（自然数）：$\mathbb{N}$
- 整数：$\mathbb{Z}$
- 有理数：$\mathbb{Q}$
- 实数：$\mathbb{R}$
- 复数：$\mathbb{C}$

且 $(\mathbb{N}^+ \text{或} \mathbb{N}^*) \subsetneqq \mathbb{N} \subsetneqq \mathbb{Z} \subsetneqq \mathbb{Q} \subsetneqq \mathbb{R} \subsetneqq \mathbb{C}$ （数 $\geq 0$、负整、分数、无理、虚数）

三、基本运算

- 并集 $\cup$：（合并、或字）
- 交集 $\cap$：（交合、且字）
- 全集、补集 $C_U$：（余积、非字）

运算性质

- 交换律：$A \cup B = B \cup A$，$A \cap B = B \cap A$
- 结合律：$(A \cup B) \cup C = A \cup (B \cup C)$，$(A \cap B) \cap C = A \cap (B \cap C)$
- 分配律：$(A \cap B) \cup C = (A \cup C) \cap (B \cup C)$，$(A \cup B) \cap C = (A \cap C) \cup (B \cap C)$

补集德·摩根定律

- 两集交集的补集等于两集补集的并集：$C_U(A \cap B) = (C_U A) \cup (C_U B)$
- 两集并集的补集等于两集补集的交集：$C_U(A \cup B) = (C_U A) \cap (C_U B)$

映射

一、概念

映射表示集合与集合间的对应关系。

- 求导是从可导函数到导函数的映射
- 函数是从数集到数集的映射（狭义）
- 线性变换是从线性空间到线性空间的映射

二、定义

设 $A$、$B$ 为非空集，若在对应法则 $f$ 下，使得每个 $x \in A$，都有唯一确定的 $y \in B$ 与之对应，则称法则 $f$ 为从 $A$ 到 $B$ 的映射。

- 记为 $f: A \to B$
- 一般表示形式为 $y = f(x)$，$x \in A$，$y \in B$
- 严格记作：$\forall x \in A$，$\exists! y \in B: y = f(x)$（唯一）
- 记 $A$ 为映射 $f$ 的定义域，记作 $D_f$
- $y$ 称为元素 $x$ 在映射 $f$ 下的像，记作 $f(x)$
- $f(x)$ 的集合称为映射 $f$ 的值域，记作 $R_f$

一般地，一个自变量仅对应一个因变量，称为单值映射；特殊地，若一个自变量对应多个因变量的集合，称为集值映射。不引起歧义下文中映射均指单值映射，集值映射将额外指明。

函数变换

该篇含部分感性理解，有空证明后将改进。

函数变换含平移、伸缩、翻折、旋转等。我们将函数变换看作是点集的映射即可。

设 $A = \{(x, y) \mid x \in D_f, y = f(x)\}$，$B = \{(x', y') \mid x' \in D_{f'}, y' = f'(x')\}$

映射 $g: A \to B$。则 $A$ 为原函数，$B$ 为函数变换后的新函数。

再记 $g(A) = B$ 为 $A \stackrel{g}{\longmapsto} B$

则有 $(x, y) \stackrel{g}{\longmapsto} (x', y')$

我们仅需知道 $x$ 与 $x'$，$y$ 与 $y'$ 的关系即可

1. 平移（左移 $m$，上移 $n$）

易知 $x' = x - m$，$y' = y + n = f'(x) + n = f(x - m) + n$

即 $(x, y) \stackrel{g}{\longmapsto} (x - m, y + n)$

用换元思想：先令 $x = x - m$，即 $f(x) \to f(x + m)$；先令 $f(x) = f(x) + n$，即 $f(x - m) \to f(x - m) + n$

(待补)2 . 推广

（1）奇函数

- $\forall x \in D$，有 $f(x) = -f(-x) \iff f(x + a) = -f(-(x + a)) = -f(-x - a) \iff f(x)$ 是奇函数
- $\forall x \in D$，有 $f(x + a) = -f(x + a) \iff f(x)$ 关于 $(a, 0)$ 对称 $\iff f(x + a)$ 是奇函数
- $\forall x \in D$，有 $f(x) + a = -[f(-x) + a] = -f(-x) - a \iff f(x)$ 关于 $(0, -a)$ 对称 $\iff f(x) + a$ 是奇函数
- 设 $g(x) = f(ax + b) + c$，则 $g(-x) = -g(x) \implies f(ax + b) + f(-ax + b) = -2c$ 
  $\iff f(ax + b) + c$ 是奇函数 $\iff f(x)$ 关于 $(b, -c)$ 对称
  $\iff f(ax + b) + f(-ax + b) = -2c \iff f(x)$ 关于 $(b, -c) \iff f(ax + b) + c$ 是奇函数

（2）偶函数

$f(ax + b) = f(-ax + b) \iff f(x)$ 关于 $x = \frac{b}{a}$ 对称 $\iff f(x + b)$ 是偶函数

3. 常见奇偶函数形式

- 奇函数：$f(x) = ax \pm \frac{b}{x} \pm cx^3 \pm d\sin x \pm e\tan x$（关于 $x$ 皆为奇次）
- 偶函数：$f(x) = a|x| \pm \frac{b}{|x|} \pm cx^2 \pm d\cos x \pm C$（关于 $x$ 皆为偶次）

且有 $|\text{奇函数}| =$ 偶函数

4. 积复合函数奇偶性（两函数有公共定义域）

- 奇 $\pm$ 奇 = 奇；偶 $\pm$ 偶 = 偶；奇 $\pm$ 偶 = 非奇非偶
- 奇 $\times$ 奇 = 偶；偶 $\times$ 偶 = 偶；奇 $\times$ 偶 = 奇（奇函数个数奇为奇，偶为偶）
- $\frac{\text{奇}}{\text{奇}} =$ 偶；$\frac{\text{偶}}{\text{偶}} =$ 偶；$\frac{\text{奇}}{\text{偶}} = \frac{\text{偶}}{\text{奇}} =$ 奇（奇 $= \frac{\text{奇}}{\text{奇}}$，偶 $= \frac{\text{偶}}{\text{偶}}$）
- 奇 (奇) = 奇；奇 (偶) = 奇；偶 (奇) = 偶；偶 (偶) = 偶

5. 奇部偶部

对于定义域关于原点对称的 $f(x)$，可以表示成"奇函数与偶函数之和"

即 $f(x) = \frac{f(x) - f(-x)}{2} + \frac{f(x) + f(-x)}{2}$（奇部 + 偶部）

6. 常见结论

- $f(x)$ 在定义区间上能取最值（或定义为闭区间），有 $f(x)_{\text{max}} + f(x)_{\text{min}} = 0$
- $f(x) = g(x) + C$，其中 $g(x)$ 为奇函数，则 $f(a) + f(-a) = g(a) + g(-a) + 2C = 2C$
- 若 $0 \in D_{\text{奇}}$，则 $f(0) = 0$

函数周期性

1. 定义

若 $f(x + T) = f(x)$，则 $f(x)$ 是周期函数，$T$ 为 $f(x)$ 的一个周期。

2. 性质

- 并不是所有周期函数都有最小正周期 $T_m$
- 周期函数定义域无界

3. 结论

- $f(x + a) = f(x + b) \implies T = |b - a|$
- $f(x + a) = -f(x) + k \implies T = 2a$
- $f(x + a) = \frac{k}{f(x)} \implies T = 2a$
- $f(x + a) = -\frac{k}{f(x)} \implies T = 2a$
- 奇函数满足 $f(a + x) = f(a - x) \implies$ 关于 $(0, 0)$ 和 $(a, 0)$ 对称 $\implies T = 4a$
- 偶函数满足 $f(a + x) = f(a - x) \implies$ 关于 $x = 0$ 和 $x = a$ 对称 $\implies T = 2a$
- 若函数关于 $x = a$ 和 $x = b$ 或 $(a, 0)$ 和 $(b, 0)$ 对称 $\implies T = 2|b - a|$
- $f(x + a) = \frac{1 - f(x)}{1 + f(x)} \implies T = 2a$；$f(x + a) = \frac{1 + f(x)}{1 - f(x)} \implies T = 4a$

4. 周期对称区分

- $f(x + a) = \pm f(x + b) \implies$ 周期性
- $f(a + x) = \pm f(b - x) \implies$ 对称性

符号相同表周期，符号不同表对称

函数的变换

1. 平移

- $g(x) = f(x) + k$（上移 $k$）
- $g(x) = f(x + h)$（左移 $h$）；$g(x) = f(x - h)$（右移 $h$）
- $g(x) = f(x) - k$（下移 $k$）

口诀：左加右减，上加下减

2. 对称

- $f(x)$ 关于 $x$ 轴对称函数 $g(x) = -f(x)$
- $f(x)$ 关于 $y$ 轴对称函数 $g(x) = f(-x)$
- 关于原点 $g(x) = -f(-x)$
- $y = a^x$ 关于 $y = x$ 对称为 $g(x) = \log_a x$

幂函数

1. 定义

$f(x) = x^a$。（$x$ 作底数，指数 $a$ 为常数，系数为 1）

2. 性质

- 在 $(0, +\infty)$ 上恒有定义，并过定点 $(1, 1)$
- 单调性：在 $(0, +\infty)$ 上，$a > 0$ 时单调递增，$a < 0$ 时单调递减
- 在 $(0, 1]$ 为"指大图低"，在 $[1, +\infty)$ 为"指大图高"
- 奇偶性：对 $f(x) = x^{\frac{m}{n}}$（$m \in \mathbb{N}^+$，$n \in \mathbb{Z}$，$m$ 与 $n$ 互质）而言：
  - $n$ 为偶数时，$f(x)$ 为偶函数
  - $m$、$n$ 都为奇数时，$f(x)$ 为奇函数
  - $m$ 为偶数、$n$ 为奇数时 $f(x)$ 为非奇非偶函数，图象只在一象限（并非所有幂函数都有奇偶性）

指数函数

1. 指数

- 根式：$\sqrt[n]{a}$
- $(\sqrt[n]{a})^n = a$（$n \geq 1$ 且 $n \in \mathbb{N}^+$）
- $\sqrt[n]{a^n} = \begin{cases} a, & n\text{为奇数} \\ |a|, & n\text{为偶数} \end{cases}$
- 分数指数幂：$a^{\frac{m}{n}} = \sqrt[n]{a^m}$（$a > 0$；$m, n \in \mathbb{N}^+$，$n \geq 1$）

2. 指数函数

- 定义：$f(x) = a^x$（$a > 0$，定义域 $\mathbb{R}$，系数为 1）
- 性质：
  - 过定点 $(0, 1)$
  - 在 $(-\infty, 0]$"底大图低"，在 $[0, +\infty)$ 底大图高
  - $a > 1$ 时，为增函数；$0 < a < 1$ 时，为减函数

对数函数

1. 对数（$a > 0$ 且 $a \neq 1$，$N > 0$）

- 定义：$a^x = N$（$a > 0$ 且 $a \neq 1$）$\iff x = \log_a N$。（$a$ 为底数，$N$ 为真数）
- 性质：
  - 负数与 0 没有对数
  - $\log_a 1 = 0$，$\log_a a = 1$，$\log_a a^k = k$，$a^{\log_a N} = N$
  - 取 $e = 2.71828\cdots$，$\log_e N$ 记作 $\ln N$
  - 记 $\log_{10} N$ 为 $\lg N$
  - $\log_a(MN) = \log_a M + \log_a N$，$\log_a \frac{1}{m} = -\log_a M$
  - $\log_a \frac{M}{N} = \log_a M + \log_a \frac{1}{N} = \log_a M - \log_a N$
  - $\log_a M^N = N\log_a M$
  - 对⑤的推广 $\log_a \left(\prod_{i=1}^N M_i\right) = \sum_{i=1}^N [\log_a(M_i)]$
  - 换底公式：$\log_a b = \frac{\log_c b}{\log_c a}$
  - $\log_a b \cdot \log_b a = 1$
  - $\log_{(a^n)} b^{m} = \frac{m}{n}\log_a b$
  - $\log_a m \cdot \log_b n = \log_a n \cdot \log_b m$
  - $a^{\log_c b} = b^{\log_c a}$

2. 对数函数

- 定义：$f(x) = \log_a x$（底数 $a > 0$ 且 $a \neq 1$，$x > 0$）
- 性质：
  - 过定点 $(1, 0)$
  - $0 < x < 1$ 时"底大图高"，$x > 1$ 时"底大图低"
  - $0 < x < 1$ 时在 $(0, +\infty)$ 减，$x > 1$ 时在 $(0, +\infty)$ 增

反函数（记函数 $f(x)$ 反函数为 $f^{-1}(x)$）（前提：$f(x)$ 是双射函数）

1. 定义

互为反函数的两个函数图象关于 $y = x$ 对称（即逆映射 $f^{-1}: B \to A$，定义域与值域互换，对应关系反转）

2. 性质

- $f(a) = b \iff f^{-1}(b) = a$
- 单调函数必有反函数，两个函数单调性相同

零点

1. 定义

使方程 $f(x) = 0$ 成立的实数 $x$ 称为函数零点（于函数是点，于实数是数）

2. 意义

方程 $f(x) = 0$ 有解 $\iff f(x)$ 有零点 $\iff f(x)$ 图象与 $x$ 轴的公共点

3. 性质

- $f(x)$ 有多个相等的实数解，称为多重零点。如 $f(x) = (x - 3)^2$，$x = 3$ 为双重零点
- 连续图象函数经过奇重零点变号，经过偶重零点不变号

4. 二分法

在 $[a, b]$ 连续的 $f(x)$，通过不断折半区间，进而得到一个零点近似值

过程：

- 定精度 $\varepsilon$，确定区间 $f(a)f(b) < 0$
- 求中点 $c = \frac{a + b}{2}$ 画函数值 $f(c)$
- 判断：
  - $f(c) = 0 \implies x_0 = c$
  - $f(a)f(c) < 0 \implies$ 令 $b = c$
  - $f(c)f(b) < 0 \implies$ 令 $a = c$
  
- 判断是否达到精度 $|a - b| \leq \varepsilon$，是则取近似值 $a$ 或 $b$，否则回到步骤（2）

5. 狭义零点存在定理

（1）定义：若 $f(x)$ 在 $[a, b]$ 上连续且 $f(a)f(b) < 0$，则 $f(x)$ 在 $(a, b)$ 上至少有一个零点。

（2）若 $f(x)$ 在 $[a, b]$ 上单调且符合（1）条件，则 $f(x)$ 在 $(a, b)$ 上有唯一零点。

6. 广义零点存在定理

（1）定义：约定无论 $f(k)$ 中 $k$ 处函数值是否有意义，是否 $\infty$，都记 $f(k)$ 为 $x = k$ 的对应值。则若 $f(x)$ 在 $(a, b)$ 上连续，且在端点上不一定有定义，且 $f(a)f(b) < 0$，那么 $f(x)$ 在 $(a, b)$ 上有零点。

十二、经典函数零点模型

1. 一次函数

$f(x) = ax + b \iff x = -\frac{b}{a}$

2. 二次函数

$f(x) = ax^2 + bx + c$，$\Delta = b^2 - 4ac$，$m = -\frac{b}{2a} = \frac{x_1 + x_2}{2}$

（1）零点数

- ① $\Delta > 0$，$x_1 = \frac{-b - \sqrt{\Delta}}{2a}$，$x_2 = \frac{-b + \sqrt{\Delta}}{2a}$
- ② $\Delta = 0$，$x_1 = x_2 = -\frac{b}{2a}$
- ③ $\Delta < 0$，$x_1 = \frac{-b - \sqrt{-\Delta}i}{2a}$，$x_2 = \frac{-b + \sqrt{-\Delta}i}{2a}$（或无实根）

（2）零点符号

- ①两不等正根 $\iff \begin{cases} \Delta > 0 \\ x_1 + x_2 = -\frac{b}{a} > 0 \\ x_1x_2 = \frac{c}{a} > 0 \end{cases}$
- ②两不等负根 $\iff \begin{cases} \Delta > 0 \\ x_1 + x_2 = -\frac{b}{a} < 0 \\ x_1x_2 = \frac{c}{a} < 0 \end{cases}$
- ③一正根一负根 $\iff ac < 0$

（3）零点分布（设已知区间 $(k_1, k_2)$ 或一点 $k$，对两根分布讨论）

- ① $x_1 < x_2 < k \iff \begin{cases} \Delta \geq 0 \\ m = -\frac{b}{2a} < k \\ af(k) > 0 \end{cases}$
- ② $k < x_1 < x_2 \iff \begin{cases} \Delta \geq 0 \\ m = -\frac{b}{2a} > k \\ af(k) < 0 \end{cases}$
- ③ $x_1 < k < x_2 \iff af(k) < 0$
- ④ $k_1 < x_1 < x_2 < k_2 \iff \begin{cases} \Delta \geq 0 \\ k_1 < m = -\frac{b}{2a} < k_2 \\ af(k_1) > 0 \\ af(k_2) < 0 \end{cases}$

3. 三次函数

$f(x) = ax^3 + bx^2 + cx + d$，$f'(x) = 3ax^2 + 2bx + c$，$\Delta' = 4b^2 - 12ac$

（1）零点数

- ① $f(x)$ 有 1 根 $\iff \Delta' \leq 0$ 或 $\begin{cases} \Delta' > 0 \\ f(x_1')f(x_2') > 0 \end{cases}$
- ② $f(x)$ 有 2 根 $\iff \begin{cases} \Delta' > 0 \\ f(x_1')f(x_2') = 0 \end{cases}$
- ③ $f(x)$ 有 3 根 $\iff \begin{cases} \Delta' > 0 \\ f(x_1')f(x_2') < 0 \end{cases}$

不等式

一、基本不等式

$a^2 + b^2 \geq 2ab$，当且仅当 $a = b$；令 $a = \sqrt{a}$，$b = \sqrt{b}$（$a, b > 0$）则 $a + b \geq 2\sqrt{ab}$，当且仅当 $a = b$

二、链式不等式

$\frac{n}{\sum \frac{1}{a_i}} \leq \sqrt[n]{\prod a_i} \leq \frac{\sum a_i}{n} \leq \sqrt{\frac{\sum a_i^2}{n}}$（$a_i > 0$ 且 $n \geq 2$）

即调和平均 $H_n \leq$ 几何平均 $G_n \leq$ 算术平均 $A_n \leq$ 平方平均 $Q_n$，当 $a_1 = a_2 = \dots = a_n$ 时取等

三、基本最值不等式

- $a + b \geq 2\sqrt{ab}$（$a, b > 0$）积定和最小
- $ab \leq \left( \frac{a + b}{2} \right)^2 \leq \frac{a^2 + b^2}{2}$（$a, b > 0$）和定积最大

四、基本不等式常见推论

1. $\frac{b}{a} + \frac{a}{b} \geq 2$（$ab > 0$）当且仅当 $a = b$
2. $\frac{b}{a} + \frac{a}{b} \leq -2$（$ab < 0$）当且仅当 $a = -b$
3. $mx + \frac{n}{x} \geq 2\sqrt{mn}$，当且仅当 $x = \sqrt{\frac{n}{m}}$（$x, m, n > 0$）
4. $mx + \frac{n}{x - a} = m(x - a) + \frac{n}{x - a} + ma \geq 2\sqrt{mn} + ma$，当且仅当 $x - a = \sqrt{\frac{n}{m}}$（$x, m, n > 0$）
5. $\frac{x}{ax^2 + bx + c} = \frac{1}{ax + \frac{c}{x} + b} \leq \frac{1}{2\sqrt{ac} + b}$，当且仅当 $x = \sqrt{\frac{c}{a}}$（$x, a, c > 0$）
6. $x(n - mx) = \frac{mx(n - mx)}{m} \leq \frac{1}{m} \left( \frac{mx + n - mx}{2} \right)^2 = \frac{n^2}{4m}$，当且仅当 $x = \frac{n}{2m}$（$m, n > 0, 0 < x < \frac{n}{m}$）

五、绝对值不等式

$||a| - |b|| \leq |a \pm b| \leq |a| + |b|$（$a$ 和 $b$ 换成向量或复数仍成立）

六、柯西不等式

$\left( \sum_{i=1}^n a_i^2 \right) \left( \sum_{i=1}^n b_i^2 \right) \geq \left( \sum_{i=1}^n a_ib_i \right)^2$

- 二元形式：$(a^2 + b^2)(c^2 + d^2) \geq (ac + bd)^2$
- 三元形式：$(a^2 + b^2 + c^2)(d^2 + e^2 + f^2) \geq (ad + be + cf)^2$

当且仅当 $\frac{a_1}{b_1} = \frac{a_2}{b_2} = \dots = \frac{a_n}{b_n}$ 时取等

推论

1. 当 $b_1 = b_2 = \dots = b_n$ 时，有 $\sqrt{\frac{\sum a_i^2}{n}} \geq \frac{\sum a_i}{n}$（$Q_n \geq A_n$），当且仅当 $a_1 = a_2 = \dots = a_n$
2. 当 $b_i = \frac{1}{a_i}$ 时，有 $\left( \sum a_i^2 \right) \left( \sum \frac{1}{a_i^2} \right) \geq n^2$，当且仅当 $a_1 = a_2 = \dots = a_n$
3. 当 $a_i, b_i > 0$ 时，有 $\sum \frac{a_i^2}{b_i} \geq \frac{\left( \sum a_i \right)^2}{\sum b_i}$，即权方和不等式，当且仅当 $\frac{a_1}{b_1} = \frac{a_2}{b_2} = \dots = \frac{a_n}{b_n}$

七、权方和不等式

$\sum_{i=1}^n \frac{a_i^2}{b_i} \geq \frac{\left( \sum_{i=1}^n a_i \right)^2}{\sum_{i=1}^n b_i}$（$a_i, b_i > 0$）

- 二元形式：$\frac{a^2}{x} + \frac{b^2}{y} \geq \frac{(a + b)^2}{x + y}$
- 变形：$\frac{a}{x} + \frac{b}{y} \geq \frac{(\sqrt{a} + \sqrt{b})^2}{x + y}$

当且仅当 $\frac{a_1}{b_1} = \frac{a_2}{b_2} = \dots = \frac{a_n}{b_n}$ 时取等

推广

$\sum_{i=1}^n \left[ \frac{(a_i)^{m+1}}{(b_i)^m} \right] \geq \frac{\left( \sum_{i=1}^n a_i \right)^{m+1}}{\left( \sum_{i=1}^n b_i \right)^m}$

（$a_i, b_i, m > 0$，当且仅当 $\frac{a_1}{b_1} = \frac{a_2}{b_2} = \dots = \frac{a_n}{b_n}$ 时取等，$m$ 称为不等式的权）

八、伯努利不等式

$\prod_{i=1}^n (1 + x_i) \geq 1 + \sum_{i=1}^n x_i$（$x_i$ 各项符号相同且 $x_i > -1$）

当 $x_i$ 各项相等，变为 $(x + 1)^n \geq 1 + nx$

九、幂平均不等式

若 $a_i > 0$，$\alpha < \beta$，则 $\left( \frac{\sum_{i=1}^n a_i^\beta}{n} \right)^{\frac{1}{\beta}} \leq \left( \frac{\sum_{i=1}^n a_i^\alpha}{n} \right)^{\frac{1}{\alpha}}$

加权形式

若 $a_i > 0$，$p_i > 0$，$\alpha < \beta$，则 $\left[ \frac{\sum (p_i a_i^\beta)}{\sum p_i} \right]^{\frac{1}{\beta}} \leq \left[ \frac{\sum (p_i a_i^\alpha)}{\sum p_i} \right]^{\frac{1}{\alpha}}$

十、切比雪夫不等式

若 $a_1 \leq a_2 \leq \dots \leq a_n$，$b_1 \leq b_2 \leq \dots \leq b_n$，则 $\left( \sum a_i \right) \left( \sum b_i \right) \leq n \left( \sum a_i b_i \right)$

十一、排序不等式

若 $a_1 \leq a_2 \leq \dots \leq a_n$，$b_1 \leq b_2 \leq \dots \leq b_n$，则
a_1b_1 + a_2b_2 + \dots + a_nb_n \geq a_{(1)}b_1 + a_{(2)}b_2 + \dots + a_{(n)}b_n \geq a_nb_1 + a_{n-1}b_2 + \dots + a_1b_n


即$[\sum a_i b_i]_{\text{正序和}} \geq [\sum a_{(i)} b_i]_{\text{乱序和}} \geq [\sum a_{n+1-i} b_i]_{\text{反序和}}$

十二、Holder 不等式

若 $a_i, b_i > 0$，$p > 1$，$\frac{1}{p} + \frac{1}{q} = 1$，则
\sum a_i b_i \leq \left( \sum a_i^p \right)^{\frac{1}{p}} \left( \sum b_i^q \right)^{\frac{1}{q}}

十三、琴生不等式

设 $f(x)$ 为 $[a, b]$ 的下凸函数，$x_i \in [a, b]$，则
\frac{\sum f(x_i)}{n} \geq f\left( \frac{\sum x_i}{n} \right)

三角函数

一、任意角

$(0, 360^\circ) \to (-\infty^\circ, +\infty^\circ)$，详略

二、弧度制

1. $(-\infty^\circ, +\infty^\circ) \to (-\infty, +\infty)$，单位 $\text{rad}$
2. 性质：

- $360^\circ = 2\pi \text{ rad}$，$180^\circ = \pi \text{ rad}$
- $1^\circ = \frac{\pi}{180} \text{ rad} \approx 0.01745$，$1 \text{ rad} = \left( \frac{180}{\pi} \right)^\circ \approx 57.29578^\circ$
- 圆心角大小 $|\alpha| = \frac{l(\text{弧长})}{r(\text{半径})}$
- $l = |\alpha|r$，周长 $K = 2r + l$，面积 $S = \frac{1}{2}|\alpha|r^2 = \frac{1}{2}lr$

三、三角函数定义

对任意角 $\alpha$，落在单位圆上坐标 $P(x, y)$，则 $\begin{cases} x = \cos\alpha \\ y = \sin\alpha \\ \frac{y}{x} = \tan\alpha \end{cases}$

1. 诱导公式

- （1）$\sin(\pi + \alpha) = -\sin\alpha$，$\cos(\pi + \alpha) = -\cos\alpha$，$\tan(\pi + \alpha) = \tan\alpha$
- （2）$\sin(-\alpha) = -\sin\alpha$，$\cos(-\alpha) = \cos\alpha$，$\tan(-\alpha) = -\tan\alpha$
- （3）$\sin\left( \frac{\pi}{2} - \alpha \right) = \cos\alpha$，$\cos\left( \frac{\pi}{2} - \alpha \right) = \sin\alpha$
- （4）$\sin\left( \frac{\pi}{2} + \alpha \right) = \cos\alpha$，$\cos\left( \frac{\pi}{2} + \alpha \right) = -\sin\alpha$
- （5）$\sin\left( \frac{3\pi}{2} - \alpha \right) = -\cos\alpha$，$\cos\left( \frac{3\pi}{2} - \alpha \right) = -\sin\alpha$

2. $\sin\alpha \pm \cos\alpha$ 与 $\sin\alpha \cos\alpha$ 之间的转化

- （1）$(\sin\alpha + \cos\alpha)^2 = 1 + 2\sin\alpha \cos\alpha$
- （2）$(\sin\alpha - \cos\alpha)^2 = 1 - 2\sin\alpha \cos\alpha$
- （3）$(\sin\alpha + \cos\alpha)^2 + (\sin\alpha - \cos\alpha)^2 = 2$

四、三角恒等变换

1. 余弦和差

- （1）$\cos(\alpha + \beta) = \cos\alpha \cos\beta - \sin\alpha \sin\beta$
- （2）$\cos(\alpha - \beta) = \cos\alpha \cos\beta + \sin\alpha \sin\beta$

2. 正弦和差

- （1）$\sin(\alpha + \beta) = \sin\alpha \cos\beta + \cos\alpha \sin\beta$
- （2）$\sin(\alpha - \beta) = \sin\alpha \cos\beta - \cos\alpha \sin\beta$

3. 正切和差

- （1）$\tan(\alpha + \beta) = \frac{\tan\alpha + \tan\beta}{1 - \tan\alpha \tan\beta} = \frac{\sin2\alpha + \sin2\beta}{\cos2\alpha + \cos2\beta}$
- （2）$\tan(\alpha - \beta) = \frac{\tan\alpha - \tan\beta}{1 + \tan\alpha \tan\beta}$

4. 二倍角

- （1）$\sin2\alpha = 2\sin\alpha \cos\alpha$
- （2）$\cos2\alpha = \cos^2\alpha - \sin^2\alpha = 2\cos^2\alpha - 1 = 1 - 2\sin^2\alpha = (\cos\alpha + \sin\alpha)(\cos\alpha - \sin\alpha)$
- （3）$\tan2\alpha = \frac{2\tan\alpha}{1 - \tan^2\alpha}$

5. 半角公式

- （1）$\sin\alpha = \pm \sqrt{\frac{1 - \cos\alpha}{2}}$
- （2）$\cos\alpha = \pm \sqrt{\frac{1 + \cos\alpha}{2}}$
- （3）$\tan\frac{\alpha}{2} = \pm \sqrt{\frac{1 - \cos\alpha}{1 + \cos\alpha}} = \frac{\sin\alpha}{1 + \cos\alpha} = \frac{1 - \cos\alpha}{\sin\alpha} = \frac{1 + \sin\alpha - \cos\alpha}{1 + \sin\alpha + \cos\alpha}$

6. 积化和差

- （1）$\sin\alpha \cos\beta = \frac{1}{2}[\sin(\alpha + \beta) + \sin(\alpha - \beta)]$
- （2）$\cos\alpha \sin\beta = \frac{1}{2}[\sin(\alpha + \beta) - \sin(\alpha - \beta)]$（$= \frac{1}{2}[\sin(\alpha + \beta) + \sin(\beta - \alpha)]$）
- （3）$\cos\alpha \cos\beta = \frac{1}{2}[\cos(\alpha + \beta) + \cos(\alpha - \beta)]$
- （4）$\sin\alpha \sin\beta = -\frac{1}{2}[\cos(\alpha + \beta) - \cos(\alpha - \beta)]$

7. 和差化积

- （1）$\sin\alpha + \sin\beta = 2\sin\frac{\alpha + \beta}{2} \cos\frac{\alpha - \beta}{2}$
- （2）$\sin\alpha - \sin\beta = 2\cos\frac{\alpha + \beta}{2} \sin\frac{\alpha - \beta}{2}$（$= 2\sin\frac{\alpha - \beta}{2} \cos\frac{\alpha + \beta}{2}$）
- （3）$\cos\alpha + \cos\beta = 2\cos\frac{\alpha + \beta}{2} \cos\frac{\alpha - \beta}{2}$
- （4）$\cos\alpha - \cos\beta = -2\sin\frac{\alpha + \beta}{2} \sin\frac{\alpha - \beta}{2}$

8. 辅助角

- （1）$a\sin\omega x + b\cos\omega x = \sqrt{a^2 + b^2} \sin(\omega x + \varphi)$（$\tan\varphi = \frac{b}{a}$）
- （2）$a\sin\omega x - b\cos\omega x = \sqrt{a^2 + b^2} \sin(\omega x - \varphi)$（$\tan\varphi = \frac{b}{a}$）

9. 万能公式

- （1）$\sin\alpha = \frac{2\tan\frac{\alpha}{2}}{1 + \tan^2\frac{\alpha}{2}} = \frac{\sin2\alpha}{2\cos\alpha}$
- （2）$\cos\alpha = \frac{1 - \tan^2\frac{\alpha}{2}}{1 + \tan^2\frac{\alpha}{2}} = \frac{\sin2\alpha}{2\sin\alpha} = \frac{1 + \tan\frac{\alpha}{2}}{1 - \tan\frac{\alpha}{2}}$
- （3）$\tan\alpha = \frac{2\tan\frac{\alpha}{2}}{1 - \tan^2\frac{\alpha}{2}} = \frac{(1 - \tan^2\alpha)\tan2\alpha}{2}$

10. 三倍角

- （1）$\sin3\alpha = 3\sin\alpha - 4\sin^3\alpha$
- （2）$\cos3\alpha = 4\cos^3\alpha - 3\cos\alpha$
- （3）$\tan3\alpha = \frac{3\tan\alpha - \tan^3\alpha}{1 - 3\tan^2\alpha}$

11. 其他

- （1）$\sin(\alpha + \beta)\sin(\alpha - \beta) = \sin^2\alpha - \sin^2\beta$
- （2）$\cos(\alpha + \beta)\cos(\alpha - \beta) = \cos^2\alpha - \sin^2\beta$
- （3）$\tan^2\alpha \sin^2\alpha = \tan^2\alpha - \sin^2\alpha$（用万能公式即可证）

平面向量

一、定义

1. 向量：我们把既有大小又有方向的量称为向量，把只有大小没有方向的量称为数量。向量用有向线段 $\overrightarrow{AB}$ 表达（但向量 $\neq$ 有向线段），它的长度/模记作 $|\overrightarrow{AB}|$。
2. 单位向量：长度等于 1 个单位长度的向量，称为单位向量，一般用 $\vec{e}$ 表示。
3. 零向量：长度等于 0 个单位长度的向量，称为零向量，用 $\vec{0}$ 表示，它的方向是任意的，与任意向量平行或垂直。
4. 平行向量/共线向量：方向相同或相反的非零向量叫做平行向量。由于零向量与任何向量平行，所以 $\vec{a} \parallel \vec{b}$，$\vec{b} \parallel \vec{c} \implies \vec{a} \parallel \vec{c}$。$\therefore \vec{a}$ 与 $\vec{b}$ 共线 $\iff$ 存在唯一一个实数 $\lambda$ 使 $\vec{b} = \lambda \vec{a}$
5. 相等向量：长度相等且方向相同的两个向量。
6. 相反向量：长度相等且方向相反的两个向量
7. 投影向量：设 $\vec{a} = \overrightarrow{AB}$，$\vec{b} = \overrightarrow{CD}$，过 $\overrightarrow{AB}$ 起点 $A$ 和终点 $B$，分别向 $\overrightarrow{CD}$ 所在直线作垂，垂足分别为 $A_1, B_1$，得 $\overrightarrow{A_1B_1}$，称上述变换为 $\vec{a}$ 向 $\vec{b}$ 投影，$\overrightarrow{A_1B_1}$ 为 $\vec{a}$ 在 $\vec{b}$ 上的投影向量。
8. 向量夹角：两向量 $\vec{a}, \vec{b}$ 的夹角 $\theta = \langle \vec{a}, \vec{b} \rangle$ 为两向量共顶点的夹角。$\theta \in [0, \pi]$

二、运算律

1. 加法：（平行四边形或三角形法则）首尾相连

- （1）交换律：$\vec{a} + \vec{b} = \vec{b} + \vec{a}$
- （2）结合律：$(\vec{a} + \vec{b}) + \vec{c} = \vec{a} + (\vec{b} + \vec{c})$

2. 减法：加相反向量
- （1）$-(-\vec{a}) = \vec{a}$，$\vec{a} + (-\vec{a}) = \vec{a} - \vec{a} = \vec{0}$

- （2）$\vec{a} - \vec{b} = \vec{a} + (-\vec{b})$

3. 乘法（数乘）：向量伸缩

（1）$|\lambda \vec{a}| = |\lambda||\vec{a}|$，$\lambda(\mu \vec{a}) = \lambda\mu \vec{a}$

（2）$(\lambda + \mu)\vec{a} = \lambda \vec{a} + \mu \vec{a}$，$\lambda(\vec{a} + \vec{b}) = \lambda \vec{a} + \lambda \vec{b}$

4. 点乘：又称内积、数量积

（1）$\vec{a} \cdot \vec{b} = |\vec{a}||\vec{b}|\cos\langle \vec{a}, \vec{b} \rangle$，$(\vec{a} + \vec{b}) \cdot \vec{c} = \vec{a} \cdot \vec{c} + \vec{b} \cdot \vec{c}$，$(\vec{a} \cdot \vec{b}) \cdot \vec{c} \neq \vec{a} \cdot (\vec{b} \cdot \vec{c})$

（2）设 $\vec{a} = x_1\vec{e_1} + y_1\vec{e_2} = (x_1, y_1)$，$\vec{b} = x_2\vec{e_1} + y_2\vec{e_2} = (x_2, y_2)$，$\vec{e_1} = (1, 0)$，$\vec{e_2} = (0, 1)$

有 $\vec{a} \perp \vec{b} \iff \vec{a} \cdot \vec{b} = 0 \iff x_1x_2 + y_1y_2 = 0 \iff |\vec{a} + \vec{b}| = |\vec{a} - \vec{b}|$

$\vec{a} \parallel \vec{b} \iff x_1y_2 - y_1x_2 = 0 \stackrel{x_2,y_2 \neq 0}{\iff} \frac{x_1}{x_2} = \frac{y_1}{y_2}$

$|\vec{a}| = \sqrt{\vec{a}^2} = \sqrt{x_1^2 + y_1^2}$

5. 叉乘：又称外积、向量积

（1）$\vec{a} \times \vec{b} = \vec{c}$，$|\vec{c}| = |\vec{a}||\vec{b}|\sin\langle \vec{a}, \vec{b} \rangle$

五、空间点、直线、平面关系

1. 四个基本事实 & 三个推论

① 过不在同一直线的三个点，有且只有一个平面。即不共线三点确定一个平面。

推论：

（1）$n$ 条直线最多把平面划分成 $1 + \frac{n(n + 1)}{2}$ 份。

（2）空间内，$n$ 个平面最多把空间分为 $\frac{n^3 + 5n + 6}{6}$ 份。

（3）相交于同一点的 $n$ 条直线最多确定 $\begin{cases} \frac{n(n - 1)}{2} \text{个平面} & (\text{任意3条不共面}) \\ \frac{(n - 3)n}{2} + 4 \text{个平面} & (\text{有3条不共面}) \end{cases}$

② 如果一条直线上两个点在同一平面，那么这条直线在这个平面内。即：$A \in l$，$B \in l$ 且 $A \in \alpha$，$B \in \alpha \implies l \subset \alpha$

③ 如果两个不重合平面有一个公共点，那么它们有且只有一条过该点的公共直线。即：$P \in \alpha$ 且 $P \in \beta \implies \alpha \cap \beta = l$ 且 $P \in l$

④ 平行同一条直线的两条直线平行

推论：

① 经过一条直线与直线外一点，有且只有一个平面

② 经过两条相交直线，有且只有一个平面

③ 经过两条平行直线，有且只有一个平面

2. 空间直线、平面的平行

① $a \not\subset \alpha$，$b \subset \alpha$，$a \parallel b \implies a \parallel \alpha$

② $a \parallel \alpha$，$\alpha \cap \beta = b$，$a \subset \beta \implies a \parallel b$

③ $a \subset \beta$，$b \subset \beta$，$a \cap b = P$，$a \parallel \alpha$，$b \parallel \alpha \implies \alpha \parallel \beta$

④ $\alpha \parallel \beta$，$\alpha \cap \gamma = a$，$\beta \cap \gamma = b \implies a \parallel b$

简记为：线线平行 $\stackrel{①}{\iff}$ 线面平行 $\stackrel{②}{\implies}$ 面面平行 $\stackrel{③}{\implies}$ 线线平行 $\stackrel{④}{\implies}$

3. 空间直线、平面的垂直

① $m \subset \alpha$，$n \subset \alpha$，$m \cap n = P$，$l \perp m$，$l \perp n \implies l \perp \alpha$

② $m \perp \alpha$，$n \perp \alpha \implies m \parallel n$

③ $a \subset \alpha$，$a \perp \beta \implies \alpha \perp \beta$

④ $\alpha \perp \beta$，$m \subset \alpha$，$\alpha \cap \beta = l$，$m \perp l \implies m \perp \beta$
