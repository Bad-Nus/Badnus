## 题目描述
>给出一个长度为 $n$ 的序列 $a_1, \ldots, a_n$，每次你可以在 **c** 的代价交换相邻的两个元素，这一操作的操作次数无限制。
>
>最终你的得分为前 $k$ 个元素之和 $(a_1, a_2, \ldots, a_k)$ 减去所花费的代价之和，请问你的得分最大可以是多少。
## 输入格式

> 输入包含一行，包含三个正整数 $n, k, c$。
> 
> 第一行给出 $n$ 个正整数 $a_1, \ldots, a_n$。
## 输出格式

> 输出包含一行，表示答案。
## 样例

### 【样例 1 输入】
```
4 2 1
2 3 5 6
```
### 【样例 1 输出】
```
7
```
### 【样例 2 输入】
```
7 3 2
2 1 3 6 10 9 7
```
### 【样例 2 输出】
```
10
```
## 数据范围

- 对于 30% 的数据， $1 < n < 10$
- 对于 60% 的数据， $1 < n < 10^5$
- 对于 100% 的数据， $1 < k < n < 3 \times 10^5$ ， $0 < c, a_i < 10^9$
