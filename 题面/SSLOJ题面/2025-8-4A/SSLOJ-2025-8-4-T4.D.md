# 题目描述
给你 $n$ 个数  
然后对于每一个数 $a_i$ ，判断是否存在一个 $a_j$ 使得 $a_i \land a_j = 0$
## 输入格式
第一行一个数 $n$  
然后 $n$ 个整数，表示 $a_i$
## 输出格式
一行， $n$ 个整数，表示答案  
如果存在，输出 $1$
否则，输出 $0$
## 样例
【样例输入】
```
6
1 2 3 5 7 9
```
【样例输出】
```
1 1 0 1 0 1 
```
