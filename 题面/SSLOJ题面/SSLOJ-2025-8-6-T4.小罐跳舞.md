## 题目描述
我们都知道，如果知道了 $A\times B\times R=C\times R$，那么可以推出 $A\times B=C$，当然前提是 $A,B,C,R$ 都是正整数。

我们还知道，如果一个事情做一次可能得不到正确答案，但是多做几次就有很大可能能够得到。

上面这两段话和题目……有很大关系，这是小罐作为一名蒟蒻给你的提示。

给出三个矩阵(行数和列数均为 $N\times N$ ) $A,B,C$ ,求 $C$ 是否等于 $A\times B$。

最多询问 $5$ 次。
## 输入格式
题目可能包含若干组数据。

对于每组数据，第一行一个数 $N$

接下来给出三个 $N\times N$的矩阵，依次为$A、B、C$ 三个矩阵。
## 输出格式
对于每组数据，输出 `Yes` 或 `No` 。
## 样例
【样例输入】
```
1
2
2
100
```
【样例输出】
```
No
```
## 数据范围与提示

对于55%的数据， $N$ 不超过 $100$；

对于100%的数据， $N$ 不超过 $1000$，矩阵中的数字保证在int范围内，数据组数不超过 $5$ 组。
