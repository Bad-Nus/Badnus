## 题目描述
给一个长度为 $N$ 的序列,请删除尽可能少的数,使新序列中满足第 $I$ 个数为 $I$ 条件的数最多.
## 输入格式
第一行有一个正整数 $n$

第二行有 $n$ 个正整数 $x_i$
## 输出格式
输出一个整数 $ans$

## 样例
### 样例输入
```
5 
1 1 2 5 4
```
### 样例输出
```
3
```
# 数据范围与提示
对于 $20\%$ 的数据: $N\le 10$  
对于 $50\%$ 的数据: $N\le 1000$  
对于 $80\%$ 的数据: $N\le 1e5$  
对于 $100\%$的数据: $N\le 5e5$
