# 题目描述

给定长度为 $n$ 的序列 $A_i$，我们将按照如下操作给 $A_i$ 排序，先找到编号最小的所在位置 $x_1$，将 $[1, x_1]$ 翻转，再找到编号第二小的所在位置 $x_2$，将 $[2, x_2]$ 翻转，如果有相同的 $A_i$，则按照输入顺序操作。输出所有的 $x_i$。

## 输入格式
第一行一个数 $n$。  
第二行 $n$ 个数 $A_i$。

## 输出格式
一行， $n$ 个数 $x_i$。

## 样例
【样例输入】  
$6$  
$3$ $4$ $5$ $1$ $6$ $2$  

【样例输出】  
$4$ $6$ $4$ $5$ $6$ $6$
### 样例解释

初始序列：  
$3$ $4$ $5$ $1$ $6$ $2$: $x_1 = 4$  

第一次翻转后：  
$1$ $5$ $4$ $3$ $6$ $2$: $x_2 = 6$  

第二次翻转后：  
$1$ $2$ $6$ $3$ $4$ $5$: $x_3 = 4$  

第三次翻转后：  
$1$ $2$ $3$ $6$ $4$ $5$: $x_4 = 5$  

第四次翻转后：  
$1$ $2$ $3$ $4$ $6$ $5$: $x_5 = 6$  

最终序列：  
$1$ $2$ $3$ $4$ $5$ $6$: $x_6 = 6$

## 数据范围与提示
对于$30\%$的数据： $n \leq 10^3$。  
对于$50\%$的数据： $n \leq 10^4$。  
对于$100\%$的数据： $1 \leq n \leq 10^5$， $-10^9 \leq A_j \leq 10^9$。
