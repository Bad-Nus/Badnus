# 定义
> 最大公约数为一组数的所有共有的因数中最大数 $gcd(a_1,a_2,...a_n)$,在不引起歧义时简记为 $(a_1,a_2...a_n)$。
> 
> 最大公约数为一组数的所有共有的倍数中最小数 $lcm(a_1,a_2,...a_n)$,在不引起歧义时简记为 $[a_1,a_2...a_n]$。
# 性质
## Gcd
1.  $$(a_1,a_2...a_n)=(\vert a_1\vert,\vert a_2\vert...\vert a_n\vert)$$
2. $$(a,b)=(b,a\mod b)$$
3. $$(a^n,b^n)=(a,b)^n$$
4. $$(a_1,a_2...a_n)=d\Rightarrow (\frac {a_1}{d},\frac {a_2}{d}...\frac {a_n}{d})=1$$
5. $$b\vert ac且(a,b)=1\Rightarrow b\vert c$$
6. $$a\vert c,b\vert c 且(a,b)=1\Rightarrow ab\vert c$$
## Lcm
1. $$[a_1,a_2...a_n]=[\vert a_1\vert,\vert a_2\vert...\vert a_n\vert]$$
2. $$[a,b]=[b,a]$$
3. $$[a^n,b^n]=[a,b]^n$$
4. $$[a_1,a_2...a_n]=d[\frac {a_1}{d},\frac {a_2}{d}...\frac {a_n}{d}]$$
5. $$[a,b,c][ab,bc,ac]=[a,b][b,c][a,c]$$
## Gcd&Lcm
1. $$(a,b)[a,b]=\vert ab\vert$$
2. $$(ab,bc,ac)[a,b,c]=\vert abc\vert$$
3. $$\frac aa$$
