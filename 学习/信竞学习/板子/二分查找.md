# 整数模板一
>**找大于等于该数的第一个数**
```cpp
int find_high(int x){    
	int l=1,r=n,mid;  
	  while (l < r){  
		mid = (l&r)+((l^r)>>1);  
		if (a[mid]>=x)  r = mid;      
		else l = mid + 1;  
	}
	return a[l];  
};  
```
>**找小于等于该数的第一个数**
```cpp
int find_low(int x){  
	int l=1,r=n,mid;  
	    while (l < r){    
		mid = (l&r)+((l^r)>>1)+1;  
		if (a[mid]<=x)  l = mid;  
		else r = mid - 1;  
	}
	return a[l];  
};  
```
