直线系又称直线束，指具有某一共同几何性质的直线集合，其方程通过引入参数描述直线族的特征。

# 方程

1. 点斜式方程

对于过点 $\left(x_0,y_0\right)$ 的直线系，方程为

$$L={\lambda}_{1} ( x-x_0 ) - {\lambda}_{2}( y-y_0 ) = 0$$

从而对过 $( x_0 , y_0 )$ 斜率为 $k=\frac{1}{m}$ 的直线，方程为

$$( y-y_0 ) = k ( x-x_0 )$$

或

$$( x-x_0 ) = m ( y-y_0 )$$

2. 两点式方程

对过 $( x_1 , y_1 )$ 和 $( x_2 , y_2 )$ 两点的直线系，由点斜式方程且斜率 $k$ 相同，有

$$
\begin{split}
&\begin{cases}
( y-y_1 ) = k ( x-x_1 )\\
( y-y_2 ) = k ( x-x_2 )
\end{cases}
\\
k
&=\frac{( y-y_1 )}{( x-x_1 )}\\
&=\frac{( y-y_2 )}{( x-x_2 )}
\end{split}
$$
