# 快读
```cpp
ll read(){
    ll x=0,f=1;
    char ch=gc();
    while(ch<'0'||ch>'9')ch=='-'&&(f=-1,0),ch=gc();
    while(ch>='0'&&ch<='9')x=(x*10)+(ch^48),ch=gc();
    return x*f;
}
```

# 递归快输
```cpp
void wrt(ll x){
    if(x<0)x=-x,pc('-');
    if(x>9)wrt(x/10);
    pc(x%10^48);
}
```

# string 快读

```cpp
string read(){
	string str="";
	char ch=gc;
	while(ch==' ' || ch=='\n' || ch=='\r')ch=gc;
	while(ch!=' ' && ch!='\n' && ch!='\r')str+=ch,ch=getchar();
	return str;
}
```
