# 简单板子

```cpp
int a[N],len,ar;//a[]存每一位,len存长度,ar表示正在进行运算的那一位
void add(int x) {
	rep(i,0,len-1)a[i]=a[i]+x+ar,ar=a[i]/10,a[i]%=10;
	if(ar)a[len++]=ar%10,ar/=10;
}
void times(int x) {
	rep(i,0,len-1)a[i]=a[i]*x+ar,ar=a[i]/10,a[i]%=10;
	while(ar)a[len++]=ar%10,ar/=10;
}
void prt() {
	for(int i=len-1;i>=0;--i)putchar(a[i]+48); //手动快输
}
```
